Discord Bot Legal Pages Template

This repository provides public legal page templates for Discord bots. It is designed for bot developers who need a Terms of Service page and a Privacy Policy page for use with Discord applications, verification, or App Directory submission.

This template includes reusable HTML files that can be hosted publicly with GitHub Pages. Each bot owner must review and edit the placeholders before using the documents.

View Terms of Service View Privacy Policy
Important: These pages are templates only. They are not automatic legal advice. Every bot owner must replace all placeholder text such as [Bot Name], [Owner Name], [Contact Email], and any data collection details before publishing.

What this repository includes

  • A reusable Terms of Service page
  • A reusable Privacy Policy page
  • A README file explaining how to use the repository
  • A simple homepage linking the legal documents

Who this is for

  • Discord bot developers
  • Developers who need public legal document links
  • Users submitting bots to Discord
  • People using GitHub Pages to host static pages

How to use

  • Copy this repository
  • Edit the placeholders in each file
  • Publish with GitHub Pages
  • Submit the public links to Discord
